# Kero IP

## 角色定义

Kero 是正文配图的固定视觉 IP。

默认每张图都要出现 Kero。Kero 不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真参与系统运转的温柔工作者。

## 外形

- 留着中分、及耳中短发的年轻男生
- 标志性的大黑圆眼（瞳孔内有白色高光圆点）
- 简化的点状鼻子
- 一条弧线代表嘴巴
- 耳朵线条圆润简练
- 内穿一件带有一字领和波浪花纹（几何针织纹理）的毛衣
- 外搭一件带有缝线细节的牛仔夹克
- 整体纯黑色线条（带有粗细轻重变化，外轮廓偏粗）
- 轮廓略微不规则，有手绘感

## 性格

- 温柔、治愈、亲切
- 很认真，但做的事有点荒诞
- 像一个低调的系统操作员
- 温和幽默，不卖萌
- 有点呆萌，但不蠢
- 像在白板草图里真的负责某个工作

## 常见职责

让 Kero 承担核心动作：

- 搬运素材
- 拉线汇聚信息源
- 卡在断点里
- 在机器里操作"判断"杆
- 变成筛选漏斗
- 切开"素材鱼"
- 盖章承接话术
- 牵着承接路径
- 举警告牌看坑
- 从洞里伸手但接不住内容
- 在旁边搬砖、搭桥、开门、分拣、记录

## 禁止

- 不要把 Kero 画成过度可爱的吉祥物
- 不要画成儿童卡通角色
- 不要给 Kero 复杂服装、表情包、闪亮眼睛
- 不要让 Kero 只是站在角落里看
- 不要让 Kero 抢走结构表达
- 不要把 Kero 画得太商业、太圆润、太精致
- 不要过度强调"可爱"，要强调"温柔但认真"

## 判断标准

如果去掉 Kero，图的核心隐喻还能完全成立，说明 Kero 太装饰了；要重写提示词，让 Kero 成为动作主体。
